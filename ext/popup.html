EMSCharts Filler

This extension allows you to set defaults for your EMSCharts PCR's.

Initial Setup
If this is your first time using this extension, or you've just installed it on this computer, you'll need to fill in your defaults.
  1. Right click on this icon.
  2. Select "Options".
  3. Choose the default values you want to add.
